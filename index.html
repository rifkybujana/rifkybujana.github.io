Making AI Accessible to Anyone, Anywhere.

Knowledge is humanity's greatest resource, and access to it should be universal and effortless. From cave inscriptions to paper to the internet, every leap in knowledge transfer has propelled civilization forward. Large Language Models are the next one. Not just tools for automation, but a fundamentally new way for people to tap into everything we've learned as a species.

I believe anyone should be able to run advanced AI locally on their own device. A student in a remote village with no reliable internet connection. A business that needs to keep its data private. No compromises, no cloud dependency. That conviction drives everything I build.

Public Projects

  • Model Memory Calculator — A single-file browser app that reads GGUF model metadata and estimates RAM/VRAM usage for any context window and KV cache quantization. Supports remote URLs via HTTP Range requests and local files. No server required.
  • SAM3.c — A pure C implementation of Meta’s Segment Anything Model 3, built for fast inference without Python or framework dependencies.
  • Kolosal AI Platform — A fully local AI runtime written from scratch in C++. Runs LLMs on any device—desktop or server—with no cloud dependency. Includes a custom tensor library, inference engine, and native UI.
  • IndoBERT QA — IndoBERT Base fine-tuned on a translated SQuAD v2.0 dataset. State-of-the-art extractive question answering for Bahasa Indonesia (2021).
