Teman Belajar

About

Teman Belajar is an AI-based application developed to help children with dyslexia to understand texts more easily by answering what we want to know from a text.

Open Source

Teman Belajar is an open source project. Which mean, the source code, model, and the notebook of this project are available for public. You can contribute to make this project better, make your own version of this project, or use this project for your own project.
